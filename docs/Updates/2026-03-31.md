### #03 修复：执行面板内容溢出无法滚动到底部

根因：`CollapsibleSection` 使用 `max-h-[2000px]` + `overflow-hidden` 做展开动画，当"生成步骤"区域内容超过 2000px 时（多节点展开输出），超出部分被截断且无法滚动。

修复（方案 A+B 组合）：
- `CollapsibleSection.tsx`：`max-h-[2000px]` 替换为 CSS `grid-rows-[0fr]`/`grid-rows-[1fr]` 过渡，内容多高都不会被截断
- `TraceStepOutput.tsx`：输出区域限高 400px + `overflow-hidden`，超出时显示渐变遮罩和"展开全部"按钮；展开后显示"收起"按钮。流式输出模式不限高

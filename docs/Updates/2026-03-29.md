# StudySolo 2026-03-29 产品更新履历

## 【面向开发者】

### scripts/ 目录结构整理

对 `scripts/` 文件夹进行了规范化整理，按职责分类归入子目录：

1. **脚本分类归档**：
   - `scripts/startup/` — 启动脚本（`start-studysolo.sh`）
   - `scripts/deploy/` — 部署脚本与生产配置（`deploy-backend.sh`、`deploy-frontend.sh`、`nginx.conf`）
   - `scripts/git/` — Git 同步脚本（保持不变）
   - `start-studysolo.ps1` 因 Windows 常用保留在 `scripts/` 根目录
2. **新增临时目录**：
   - `scripts/logs/` — 运行日志、调试记录存放位置
   - `scripts/temp/` — 临时输出文件存放位置
   - `scripts/sandbox/` — 临时测试脚本存放位置
   - 三个目录均通过 `.gitkeep` + `.gitignore` 保持目录结构但不追踪内容
3. **路径修复**：移动后的 shell 脚本中项目根目录定位已从 `$0/..` 更新为 `$0/../..`
4. **新增 `scripts/README.md`**：完整记录目录结构、脚本说明、端口约定和临时目录使用规范

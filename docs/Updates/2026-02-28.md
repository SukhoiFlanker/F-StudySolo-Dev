#### v0.1.013 🏗️ 拥抱 Git Submodule 共享模块架构

为解决独立在 StudySolo 仓库中开发的 AI 助手因缺乏全局上下文导致的数据表名称误用（如 `users` 与 `workflows` 错误引用），引入了全新的联合防御架构：
- 废除原有的 Git Subtree 机制，采用 Robocopy 增量克隆代替双向代码流动，彻底避免历史合并冲突。
- 引入外挂 Git Submodule `shared/` (`1037solo-shared` 仓库)，强制引入 `AGENTS.md` 边界隔离上下文。
- 引入跨项目的强约束 TypeScript 定义 `shared/src/types/database.ts`，彻底防止 AI 在编写 Supabase 查询语句时猜错表名或无视 `ss_` 前缀。

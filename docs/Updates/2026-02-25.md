### 【面向开发者】

#### v0.1.005 🏗️ 工作流管理与画布体系构建
- **界面架构**：实现了 Dashboard Shell 三栏布局（包含响应式 Sidebar、Navbar 和 MobileNav）。
- **状态与视图**：接入 Zustand 集中状态库统管节点边线数据，采用 `@xyflow/react` 实现核心无限画布引擎。
- **健壮性同步**：实现 UI防抖实时保存、IndexedDB 本地缓存、云端延迟 PUT 的三层自动存储降级与冲突恢复策略。
- **数据流转**：基于 Pydantic 完成后端工作流 CRUD API 接口，加入严苛的数据隔离与 Schema 校验机制。

#### v0.1.006 🤖 智能路由与两段式AI调度引擎
- **配置系统**：搭建基于 YAML 的多平台模型配置中心，无硬编码支持 8 大平台灵活路由分配。
- **容灾调度**：设计包含执行优先级、降级链（Fallback Chains）的高可用智能模型路由接口（ai_router）。
- **流程生成**：实现基于 `qwen3-turbo` 的自然语言两段式（分析解析+画布生成）节点拓扑构建服务，支持 JSON Schema 强制验证失败重试与 Prompt 注入拦截防护。

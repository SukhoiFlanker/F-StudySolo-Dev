# 快速开始

欢迎来到 StudySolo！本指南将帮助你在 5 分钟内创建第一个 AI 驱动的学习工作流。

## 前置条件

在开始之前，请确保你已：

- [ ] 注册 StudySolo 账号
- [ ] 完成邮箱验证
- [ ] 登录后进入工作台

## 创建第一个工作流

### 步骤 1：新建工作流

点击工作台左侧的「新建工作流」按钮，输入你的学习目标：

```
我想学习 Python 数据分析，为期 3 个月
```

### 步骤 2：AI 生成学习路径

点击「生成工作流」按钮，AI 将：

1. 分析你的学习目标
2. 拆解知识点结构
3. 生成完整的学习节点图

### 步骤 3：查看节点图

生成完成后，你将看到：

- **输入节点**：接收学习目标
- **AI 分析节点**：拆解知识框架
- **规划节点**：生成学习路径
- **资源节点**：推荐学习资料
- **输出节点**：完整学习计划

### 步骤 4：运行工作流

点击「运行」按钮，AI 将执行整个学习规划流程，最终输出：

- 阶段性学习目标
- 推荐学习资源
- 预计完成时间

## 常见问题

### Q: 生成的节点图可以修改吗？

可以。你可以手动拖拽调整节点位置，添加或删除节点，自定义学习路径。

### Q: 支持哪些类型的学习目标？

支持各类技能学习，包括编程语言、专业课程、考证准备、兴趣爱好等。

# 创建工作流

本文档详细介绍 StudySolo 工作流系统的核心概念与操作方法。

## 什么是工作流

工作流是由多个 AI 节点组成的学习路径，每个节点负责一个具体的学习任务。节点之间通过连线形成完整的学习规划流程。

### 工作流生命周期

```
输入目标 → AI 分析 → 知识拆解 → 路径规划 → 资源推荐 → 输出计划
```

## 节点类型详解

### 基础节点

| 类型 | 说明 | 用途 |
|------|------|------|
| **用户输入** | 接收用户的学习目标 | 工作流起点 |
| **AI 分析** | 分析知识点结构 | 理解学习需求 |
| **AI 规划** | 生成学习路径 | 制定学习计划 |
| **资源推荐** | 推荐学习资料 | 提供学习资源 |
| **总结归纳** | 输出学习摘要 | 整理学习要点 |

### 高级节点

- **条件分支**：根据用户水平调整路径
- **循环节点**：支持重复练习场景
- **评估节点**：检测学习效果

## 创建方式

### 方式一：AI 自动生成（推荐）

1. 点击「新建工作流」
2. 输入学习目标描述
3. 点击「生成工作流」
4. AI 自动构建完整节点图

### 方式二：手动搭建

1. 从节点库拖拽所需节点
2. 配置每个节点的参数
3. 连接节点形成流程
4. 保存并运行测试

## 工作流编辑技巧

### 画布操作

- **拖拽**：移动节点位置
- **连线**：点击节点边缘，拖拽到目标节点
- **缩放**：鼠标滚轮或双指捏合
- **删除**：选中节点按 Delete 键

### 节点配置

每个节点支持以下配置：

- **输入参数**：定义节点接收的数据
- **输出格式**：设置节点返回的结构
- **AI 模型**：选择执行任务的模型
- **超时设置**：控制节点执行时间

## 运行与调试

### 运行工作流

点击「运行」按钮，系统将：

1. 按拓扑顺序执行节点
2. 显示每个节点的执行状态
3. 输出最终结果

### 调试技巧

- 查看节点输出日志
- 使用「单步执行」模式
- 检查节点间数据传递

## 最佳实践

- **从简单开始**：先用 AI 生成基础框架，再手动优化
- **模块化设计**：将复杂流程拆分为子工作流
- **测试覆盖**：运行前检查所有节点配置
- **文档注释**：为复杂节点添加说明

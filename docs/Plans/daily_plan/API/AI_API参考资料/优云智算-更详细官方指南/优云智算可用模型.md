# 优云智算 (Compshare) 可用模型与计费指南

> **基于 UCloud 独立算力的模型聚合平台，提供高性能、高性价比的模型接口服务。**

## 一、 限流说明

*   **默认限制**：50 RPM (每分钟请求数)
*   *如需更高额度，请联系客服申请。*

## 二、 计费规则

优云智算采用“积分倍率”计费模式，积分消耗计算公式如下：

**积分消耗** =（输入 Token × 输入倍率）
           +（缓存创建 Token × 缓存创建倍率）
           +（缓存命中 Token × 缓存命中倍率）
           +（输出 Token × 输出倍率）

## 四、 接入建议

1.  **容灾降级**：作为 StudySolo 的 **第 2 优先级** 路由，在七牛云服务异常时，建议优先通过优云智算调用同名模型（如 DeepSeek-V3.2, Kimi）。
2.  **缓存优化**：对于频繁访问的 System Prompt 或长文本上下文，利用 Claude/MiniMax 的 **缓存读取倍率** (通常仅为输入倍率的 1/10) 可大幅降低运行成本。
3.  **Base URL**：`https://api.compshare.cn/v1`

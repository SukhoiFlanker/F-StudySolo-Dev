# 🤖 AI 编程总结（Vibe Coding）

所有 AI 编程后的总结都放在这里，并且按照功能分类。

## 命名规范

文件名优先使用中文，例如：
- `工作流引擎实现.md`
- `AI双模型路由.md`
- `Supabase认证集成.md`
- `共享Supabase数据库迁移.md`

## 编写格式

每篇总结包含以下内容：

```markdown
# 标题

> 📅 日期：YYYY-MM-DD  
> 🤖 AI 工具：Copilot / Claude / Cursor / Gemini  
> 📌 关联版本：vX.Y.Z

## 背景
解决什么问题，为什么需要这个功能。

## 实现过程
与 AI 协作的关键步骤、Prompt 策略。

## 核心代码
关键代码片段与解释。

## 踩坑记录
遇到的问题与解决方案。

## 经验总结
可复用的 AI 编程技巧与最佳实践。
```
